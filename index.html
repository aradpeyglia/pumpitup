Co-browsing class tests

Class: sm_cobrowsing_masked_field — Use this to test masking: the field is in a co-browse–sensitive area; the value should be obscured for the operator (e.g. password-style masking), not shown in plain text.

Class: sm_cobrowsing_hidden_field — Use this to test hiding: this block should be excluded or hidden from the co-browse view so the operator does not see it at all.

Class: sm_cobrowsing_disabled_field — Use this to test disabled: the area is non-interactive for co-browse; the visitor may still see it locally, but the session should not treat it as a live form field to mirror.
